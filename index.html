Lección 1

Introducción al Lenguaje

C++ es un lenguaje de programación de propósito general conocido por su rendimiento, eficiencia y control sobre el hardware. Soporta paradigmas procedurales y orientados a objetos, lo que lo hace versátil para una amplia gama de aplicaciones, desde programación de sistemas hasta desarrollo de videojuegos. C++ se basa en el lenguaje C, añadiendo características como clases, plantillas y manejo de excepciones.

¿Qué es C++?

C++ es un lenguaje de programación poderoso y de propósito general. Es una extensión del lenguaje C que añade características como la programación orientada a objetos, la cual permite estructurar el código en componentes reutilizables. C++ es conocido por su alto rendimiento, eficiencia y control sobre los recursos del sistema, lo que lo hace adecuado para una amplia gama de aplicaciones, incluyendo desarrollo de videojuegos, sistemas operativos y computación de alto rendimiento.

¿Por qué usar C++?

C++ es un lenguaje de programación de propósito general conocido por su rendimiento y control sobre los recursos del sistema. Combina características de alto y bajo nivel, permitiendo a los desarrolladores escribir código eficiente para una amplia gama de aplicaciones. Esto incluye sistemas operativos, desarrollo de videojuegos, sistemas embebidos, computación de alto rendimiento y mucho más.

C vs C++

C y C++ son ambos lenguajes de programación, con C++ evolucionando a partir de C. C es un lenguaje procedimental enfocado en la programación estructurada mediante funciones. C++ expande C añadiendo características de programación orientada a objetos (POO) como clases, herencia y polimorfismo, permitiendo un diseño de software más complejo y modular. C++ también introduce sobrecarga de operadores, plantillas y manejo de excepciones, ofreciendo mayor flexibilidad y abstracción en comparación con C.
C
  • Paradigma procedimental
  • Estructurado en funciones
  • Sin soporte nativo de POO
  • Menor abstracción
  • Manejo manual de memoria
C++
  • Multi-paradigma (POO, genérico)
  • Clases y herencia
  • Polimorfismo y encapsulación
  • Plantillas y genéricos
  • Manejo de excepciones
Lección 2

Configuración de tu Entorno

Configurar tu entorno en C++ implica preparar tu computadora con las herramientas y el software necesarios para escribir, compilar y ejecutar programas C++. Esto típicamente incluye instalar un compilador de C++ (como GCC o Clang), un entorno de desarrollo integrado (IDE) o editor de texto, y potencialmente un sistema de compilación para gestionar proyectos más grandes. Una configuración adecuada es clave para un flujo de desarrollo fluido y eficiente.

Instalando Code::Blocks

Code::Blocks es un IDE gratuito que incluye todo lo necesario para programar en C++. Viene con el compilador GCC integrado, por lo que no necesitas instalar herramientas adicionales. Es la opción perfecta para principiantes debido a su interfaz sencilla y configuración automática.
1
Descargar: Ve a codeblocks.org y descarga la versión con compilador GCC para Windows.
2
Instalar: Ejecuta el instalador y sigue los pasos. Asegúrate de seleccionar la opción que incluye el compilador MinGW.
3
Configurar: Al abrir Code::Blocks por primera vez, selecciona el compilador GNU GCC Compiler como predeterminado.

IDE Recomendado: Code::Blocks

Code::Blocks es un IDE gratuito y de código abierto especialmente diseñado para el desarrollo en C++. Es la herramienta perfecta para principiantes debido a su interfaz intuitiva y su facilidad de configuración. Incluye compilador integrado, depurador visual, resaltado de sintaxis y autocompletado, todo lo que necesitas para empezar a programar en C++ sin complicaciones.
